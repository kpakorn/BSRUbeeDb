# BSRU Bee Specimen Catalog Website

This is a static Github site for browsing and searching a historical museum bee specimens cataloguing under the Bee and Spider Research Unit (BSRU), Chulalongkorn University, Thailand. The catalog was retrieved from a CSV database. With helps of 

## Overview

The website is intended for museum, natural history, biodiversity, and collection-based datasets, especially historical bee specimen records. Some records were already hosted under GBIF pages including two large dataset as follow : [Database and digitization of bees in Thailand](https://www.gbif.org/dataset/cb0ab16c-7589-4a44-80d0-30bae8c952ef) , and [Database and digitization of bees from TIGER project (Thailand)](https://www.gbif.org/dataset/b4320ef2-9abb-4f3c-a3b7-33120fdbb715)

## Main Features

- Static website
- Loads specimen records directly from a CSV file
- Clean header, sidebar, and main table layout
- Search and filter tools in the left panel
- Multi-word search support
